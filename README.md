## Getting Started (Windows)
Open command line
```shell
pip install flask
pip install pandas
pip install xlrd
```

Then navigate to the project
```shell
cd C:\Users\Name\Desktop\CalPal
```

Run app.py
```shell
cd project
python app.py
```
